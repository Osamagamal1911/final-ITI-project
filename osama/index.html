Welcome to Osama Gamal's Final ITI Project

Project Overview

This is Osama Gamal's final ITI project. The project involves the use of two Jenkins pipelines:

  1. Jenkins Pipeline 1: Creating Infrastructure on Google Cloud Platform using Terraform.
  2. Jenkins Pipeline 2: Deploying the Application on the Google Kubernetes Engine (GKE) Cluster.

Technologies Used

  • Jenkins
  • Terraform
  • Docker
  • Kubernetes (K8s)
  • Google Cloud Platform (GCP)

Project Details

For this project, I implemented two Jenkins pipelines to automate the deployment process:

  1. Jenkins Pipeline 1: Infrastructure Creation

    This pipeline is responsible for creating the infrastructure on Google Cloud Platform (GCP) using Terraform. It automates the provisioning of necessary resources such as virtual machines, networks, and storage to set up the foundation for our application.

  2. Jenkins Pipeline 2: Application Deployment

    This pipeline deploys our application on the Google Kubernetes Engine (GKE) cluster. It automates the process of containerizing the application using Docker and managing its deployment on GKE. This ensures that our application is running efficiently and can scale as needed.

This project leverages the power of automation and cloud computing to streamline the process of infrastructure setup and application deployment, making it more efficient and reliable.
